+++ author = "levon" published = 2009-10-29T18:31:58.000Z slug = "2009-10-29-enabling-xvm-on-opensolaris" categories = [] title = "Enabling xVM on OpenSolaris" +++ Another significant usability improvement that landed in build 126 is Gary and Bill's work on enabling Xen. Now, running xVM should be as simple as: 

# pkg install xvm-gui
# echo 'set zfs:zfs_arc_max = 0x10000000' >>/etc/system # yes, you still need this, sadly
# svcadm enable -r milestone/xvm
# reboot

There's also a new Visual Panel for doing this if you prefer a graphical method. More in the flag day message.
